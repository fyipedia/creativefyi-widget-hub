# creativefyi-widget-hub

Family-level widget docs site for the **Creative FYI** ecosystem — hosts [widget.creativefyi.com](https://widget.creativefyi.com).

## Live site

[https://widget.creativefyi.com](https://widget.creativefyi.com) — landing page showcasing all 5 sites:

- [colorfyi-embed](https://www.npmjs.com/package/colorfyi-embed) — [widget.colorfyi.com](https://widget.colorfyi.com)
- [fontfyi-embed](https://www.npmjs.com/package/fontfyi-embed) — [widget.fontfyi.com](https://widget.fontfyi.com)
- [emojifyi-embed](https://www.npmjs.com/package/emojifyi-embed) — [widget.emojifyi.com](https://widget.emojifyi.com)
- [symbolfyi-embed](https://www.npmjs.com/package/symbolfyi-embed) — [widget.symbolfyi.com](https://widget.symbolfyi.com)
- [unicodefyi-embed](https://www.npmjs.com/package/unicodefyi-embed) — [widget.unicodefyi.com](https://widget.unicodefyi.com)

## Family

Part of the [Creative FYI](https://creativefyi.com) family, hosted under [FYIPedia](https://fyipedia.com).

## How this is deployed

- Single static HTML page in `index.html`
- Served via GitHub Pages from this repo's `main` branch
- Custom domain `widget.creativefyi.com` (CNAME in this repo + DNS at Cloudflare)
- HTTPS enforced via Let's Encrypt automatic provisioning
